
## \

原样输出带有特殊字符，在其前添加\使其变成普通字符
```bash
$echo $date

$
#$带有特殊含义，输出$date不会有任何结果

$echo \$date
$date

  
$echo \\  #\本身也是特殊字符
\
```

还可以表示一些不可打印的字符

- `\a`：响铃
- `\b`：退格
- `\n`：换行
- `\r`：回车
- `\t`：制表符

## 单引号''

单引号用于保留字符串的字面含义，各种特殊字符在单引号''里边都会变成普通字符串，使bash拓展，变量引用，算数运算和子命令都失效

```bash
  
$ echo '*'
*

$ echo '$USER'
$USER

$ echo '$((2+2))'
$((2+2))

$ echo '$(echo foo)'
$(echo foo)
```

## 双引号“ ”

双引号相较于单引号更加宽松，三个特殊字符除外：美元符号（`$`）、反引号（`` ` ``）和反斜杠（`\`）
