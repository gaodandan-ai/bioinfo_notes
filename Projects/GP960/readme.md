# GP960-growth-screen-pipeline

High-throughput growth curve screening data analysis pipeline for GP960 system. This project aims to automate the processing and analysis of microbial high-throughput growth screening experimental data generated by the GP960 system.

## Introduction

This pipeline is designed to process growth curve data from Gene Overexpression (GOE) or mutant library screening experiments. Currently, it primarily supports the **Single Replicate** and **Triplicate Replicate** experimental design analysis workflow.

Key features include:
1.  **Raw Data Processing**: Extract growth data from raw Excel files and reorganize them according to experimental conditions and plate layouts.
2.  **Data Cleaning**: Filter out unavailable genes and abnormal growth data (low OD).
3.  **Fitness Analysis**: Calculate growth parameters (growth rate, AUC, final OD, etc.) to evaluate the relative fitness of mutant strains under stress conditions.
4.  **Visualization**: Plot growth curves and tolerance metric distributions.

### Single Replicate Workflow
### 1. Raw Data Processing (Pre-processing)
Convert raw Excel data from the machine into standardized CSV format.

```bash
python src/data_processing/01.pp_rawdata.py \
    --rawdata_dir /path/to/raw_data \
    --plate_file /path/to/plate-gene-mapping.xlsx \
    --result_dir /path/to/output/01.ppraw_data \
    --c1_label control \
    --c2_label stress
```

**Arguments:**
- `--rawdata_dir`: Directory containing raw `.xlsx` data files.
- `--plate_file`: Mapping file for plate wells and genes (`plate-gene-mapping.xlsx`).
- `--result_dir`: Output directory.
- `--c1_label`: Label for condition 1 (e.g., `control` or `nonstress`).
- `--c2_label`: Label for condition 2 (e.g., `stress`).

### 2. Data Cleaning

Filter out invalid genes and low-growth data, and calculate the average of control groups.

```bash
python src/data_processing/02.data_cleaning.py \
    --input_dir /path/to/output/01.ppraw_data \
    --output_dir /path/to/output/02.cleaned_data \
    --mapping_file /path/to/plate-gene-mapping.xlsx \
    --threshold 1.0
```

**Arguments:**
- `--input_dir`: Directory generated in the previous step (`01.ppraw_data`).
- `--threshold`: OD threshold; genes with max OD below this value will be excluded (default recommended: 1.0).

### 3. Fitness Calculation

Calculate Relative Fitness (RF) and screen for tolerant genes.

```bash
python src/data_processing/03.calculate_fitness.py \
    --input_dir /path/to/output \
    --output_dir /path/to/output/03.stress_tolerance_analysis \
    --rf_threshold_stress 1.05 \
    --comprehensive_rf_threshold_stress 1.05
```

**Arguments:**
- `--input_dir`: Parent directory containing `02.cleaned_data`.
- `--rf_threshold_stress`: Screening threshold for relative fitness of final OD under stress conditions.
- `--comprehensive_rf_threshold_stress`: Screening threshold for comprehensive score.

### 4. Plotting

Batch plot growth curves for tolerant genes.

```bash
python src/data_processing/04.plot_gene_od.py \
    --input_dir /path/to/output \
    --analysis_dir /path/to/output/03.stress_tolerance_analysis \
    --output_dir /path/to/output/04.gene_growth_curves
```

**Arguments:**
- `--input_dir`: Parent directory containing `02.cleaned_data`.
- `--analysis_dir`: Analysis result directory generated in the previous step (used to read the list of screened genes).
